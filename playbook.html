Introduction to HTML:

HTML stands for Hypertext Markup language.

Hypertext refers to the pieces of text which can link to other documents in the website. These pieces of text are hypertext or hyperlinks, and they are the foundation of an HTML website.

Markup is done through HTML tags.

Anatomy of an HTML tag:

The first part of an HTML tag is called the HTML element, the second part is called an HTML attribute.

An HTML attribute gives more information to the browser to specify modifications to an HTML element. The HTML attribute comes after the main HTML element and is separated from the element by a single space.

<hr> = horizontal rule tags. This will allow us to create horizontal lines. It's self-closing and measured by 'pixels high'. Example: <hr size="3"> means the hr element has an attribute of size="3" pixels high.


What is the HTML Boilerplate?

Boilerplate: what you start off with when you create any new HTML file. Includes a <!DOCTYPE> declaration, the <html>,<head>,<meta>,<title>,</title>,*</head>,<body>,</body>,</html> elements. *There are more elements which will be added to the <head> </head> section later.

<head> element is an area where important information about our website will be placed, but won't be displayed to the user. It includes things that will help the website render in the browser correctly. <head> requires </head>.

<meta> is a tag for the character set encoding of the website. This ensures that the characters that you're using on your website get displayed correctly. The charset="UTF-8" attribute is one of the most common character sets, and we can use it as our default. <meta> is self-closing.

<title> is what gets displayed in the tab bar of your browser and should be included in the <head> section of your website. <title> requires a </title>.

<body> is where all the content of the website goes. <body> eventually closes with </body>.

Nesting and Indentation are very important in coding. Think of the indentations as elements being sandwiched between two other elements (ex: opening and closing head tags, or body tags, or even the HTML tags themselves).

To create a boilerplate in VS Code you can use a *shortcut: Type an !, hit enter on the first selection and it will automatically insert the boilerplate.
*This only works if you have created a .html website.

HTML Links and Anchors

<a href> includes an 'a' element with a 'hyperlink reference' (known as href) attribute. The href will then contain a link destination - a URL or a URL fragment that the hyperlink points to. Before the 'a' tag gets closed it includes 'link text' which, when clicked, will take you to another website.
